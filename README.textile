h1. Static Site Generator for "Gollum":http://github.com/github/gollum

Generate a static HTML site based on a Gollum wiki.

h2. Dependencies

Gollum - gem install gollum
Liquid - gem install liquid

h2. Usage

1. Place the Rakefile in your Gollum wiki repository
2. Run
<notextile><pre>
rake generate
</pre></notextile>

The static site is output to the _site directory.

h2. Layouts

Wiki pages are rendered using the file "layout.html" if present.

Layouts are loaded recursively from the wiki pages parent directory. For example:

layout.html
Home.md
sub_dir/
	layout.html
	Page1.md
sub_dir2/
	Page2.md

The "Page1.md" page will use "sub_dir/layout.html".
The "Page2.md" page will use "layout.html".
